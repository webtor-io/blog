AI Recommendations, 11 Languages & 4K Fixes ✨

April 21, 2026 What's new

A big update this time — Webtor now recommends movies with AI, speaks 11 languages, and streams 4K without freezing.

AI Recommendations in Discover

The Discover page now features AI-powered movie recommendations. Just type what you’re in the mood for — something like “suggest a fresh comedy for tonight” or “sci-fi with a strong female lead” — and the system picks movies for you, complete with posters, descriptions, and info about cast and directors.

Every recommendation set includes fresh releases, so you always see something new — not just the same old classics.

If you’re browsing in Spanish or German, recommendations come in your language too.

11 Languages

Webtor used to be English-only. Now it supports 11 languages: English, Russian, Spanish, German, French, Brazilian Portuguese, Italian, Polish, Turkish, Dutch, and Czech.

It’s not just translated buttons — the whole experience adapts to your language:

  • Movie titles and descriptions are shown in your language
  • Error messages make sense instead of showing raw English text
  • Dates and times like “6 days ago” display in your language too

Webtor detects your browser language automatically. You can also switch manually at any time.

Better on Mobile

We went through every page and improved the mobile experience:

  • More room for content — decorative elements hide on small screens so you see what matters
  • Sticky tabs — Library and Discover tabs stay visible as you scroll
  • Easier to tap — buttons and interactive elements are properly sized for touch
  • Readable file names — torrent names now wrap to two lines instead of cutting off mid-word

4K Streaming — No More Freezes

4K playback was freezing even on fast connections. We found and fixed the bottleneck — video segments now load much faster, and the player recovers quickly if something hiccups on the backend.

If you tried 4K before and gave up — give it another shot.

Improved Stability

A few things you won’t see directly but will feel:

  • Large torrents stream better — fixed memory issues that could cause interruptions on big files
  • Vault files no longer get stuck — reworked how background jobs are managed, so saved files process reliably
  • Magnet links work more consistently — fixed crashes that sometimes prevented magnet-to-torrent conversion
  • Automatic retry — if a connection drops mid-download to Vault, it picks up where it left off instead of failing

Thanks for using Webtor. More coming soon.
